###  Manual Texture Swapping

To change GUI style:

1. Go to `/Textures/`
2. Replace contents with files from:
   - `/texture_pack/vaos/`
   - `/texture_pack/green/`
   - `/texture_pack/stock/`
3. Restart game

Rollback anytime by restoring original folder.
